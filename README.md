# StockSee（stock-see）

面向中文场景的股票分析助手：用自然语言提问，系统结合可配置的模型与数据能力，给出解读与参考信息。

---

## 能做什么

- **对话分析**：围绕 A 股标的（代码或常见简称）回答行情、财务与新闻等相关问题。  
- **上下文增强**：可结合会话与本地记忆，便于连续追问与回顾。  
- **可扩展数据与工具**：通过配置接入行情、资讯、知识库等能力（具体以你部署时的配置为准）。

---

## 适合谁

个人投资者、量化/产品原型验证、学习「对话式投研助手」场景的开发者。  
输出仅供研究与学习参考，**不构成投资建议**。

---

## 快速开始（概要）

1. 准备 **Go** 运行环境。  
2. 在仓库根目录准备配置文件（可参考 `config/stock.example.json` 复制为 `config/stock.json`），填入对话模型等必要项。  
3. 启动主服务后，通过 HTTP 接口使用（默认监听 **`http://localhost:8080`**，具体路由以项目代码为准）。

详细参数、可选依赖（如 Redis、嵌入服务）与运维方式，以你实际部署环境为准；本文不展开实现细节。
